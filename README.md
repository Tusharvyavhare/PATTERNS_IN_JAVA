# ☕ Java Basics Collection

A beginner-friendly collection of **Java programs** covering fundamental programming concepts, pattern printing, loops, conditions, functions, arrays, and problem-solving examples.

This repository is designed for students who are learning Java from scratch and want to strengthen their programming fundamentals through practical examples.

# ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Java-Basics.git
```

### 2. Navigate to the project

```bash
cd Java-Basics
```

### 3. Compile

```bash
javac FileName.java
```

### 4. Run

```bash
java FileName
```

Example:

```bash
javac half_pyramid.java
java half_pyramid
```
